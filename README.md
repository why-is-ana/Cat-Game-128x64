# Cat-Game-128x64
<img src="imagenes/gato_00.jpg" alt="imagen de un gato" width="700"><br><br>
Cat Game o Juego del Gato, es un proyecto simple, cuya finalidad es que se puedan dise√±ar imagenes para un display OLED de 1.9'', consta de dos botones y cada uno de ellos afecta la imagen en general. La imagen es de un gato que bajara una mano dependiendo del boton que se presione. El objetivo es que se entienda el concepto de como se dibujan las imagenes en las pantallas y como gestionar la creacion de esta a traves de herramientas de calculo como lo pueden ser Excel o GoogleSheets.<br>
